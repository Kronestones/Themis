## How to Run

```bash
python main.py              # Watch mode — continuous scanning
python main.py --scan       # Single scan right now
python main.py --status     # Detection status
python main.py --export FILE    # Export detection log
python main.py --rights SITUATION   # Know Your Rights
python main.py --explain TOPIC      # Plain language explainer
python main.py --watchdog   # Auto-restart mode
python main.py --verify     # Verify audit log integrity
```
